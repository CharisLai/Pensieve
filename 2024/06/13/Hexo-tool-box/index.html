為何原本有安裝過hexo-cli
現在bash卻找不到

  • 版本依賴管理 (解答)
    昨日晚上為了在本地架設某個網頁，有更換過node.js版本使用(node.js v16.20.2)
    而先前使用的版本為(v18.17.1)，因此會有出現bash command not found的狀況
    各套件版本使用的重要性，像是版本相容性問題是蠻常會遇到的挑戰，軟體隨時間推陳出新，與時俱進是軟體工程師的日常工作。

  • 延伸思考 什麼是cli ?
    命令列介面 CLI Command Line Interface
    文字指令，與電腦作業系統互動的命令。CLI 在預設Shell下進行操作，該Shell位於作業系統和使用者之間。
